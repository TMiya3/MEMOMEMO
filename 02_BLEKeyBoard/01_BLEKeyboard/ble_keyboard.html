BLEキーボード

26/02/08作成

ESP32をBLEキーボードにする
まずは1秒ごとにaを打ち込むキーボードにする

ライブラリインストール

ESP32をBLEキーボードにするライブラリを入れる
参考:ライブラリのリポジトリ
zipをここ からダウンロード
インストール方法はリポジトリを見ると書いてある

zipライブラリを追加する

エクスプローラが開くのでダウンロードしたzipを選択

ソースコード

			
#include <BleKeyboard.h>

BleKeyboard bleKeyboard("ESP32 Keyboard");

void setup() {
  Serial.begin(115200);
  Serial.println("Starting BLE work!");
  bleKeyboard.begin();
}

void loop() {
  if (bleKeyboard.isConnected()) {
    bleKeyboard.print("a");
  }
  delay(1000);
}

接続

ESP32に書き込む
通電するとbluetooth機器として見つけられるので接続する

接続したとたんaが入力され始めるので注意
